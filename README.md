# [POC] Containerized Test Automation

## Pre-requisites
Install docker and docker desktop

## Building Docker Image
Simply run the following command: 

```
docker build .
```

## Build/Run Container and Run Automated Tests
Simply run the following command:
```
docker compose up
```
